Documentation service d'aide api/maths

Bienvenue sur la page de d'aide api/maths voici les services offerts: veuillez utiliser la variable 'op' pour vos requêtes, 'x' et 'y' pour les opérations et 'n' pour la factorisation et les services de nombre premiers
pour une addition: '+'
pour une soustraction: '-'
pour une multiplication: '*'
pour une division: '/'
pour un modulo: '%'
pour un factoriel: '!'
pour savoir si un nombre est premier : 'p'
pour savoir la position d'un nombre premier : 'np' Merci et bonne journée!
