Documentation service d'aide api/maths

Bienvenue sur la page de d'aide api/maths

voici les services offerts:



veuillez utiliser la variable 'op' pour vos requêtes, 'x' et 'y' pour les opérations et 'n' pour la factorisation et les services de nombre premiers

pour une addition: '? op = + & x = number & y = number return {"op":"+","x":number,"y":number,"value": x + y}'

pour une soustraction: '? op = - & x = number & y = number return {"op":"-","x":number,"y":number,"value": x - y}'

pour une multiplication: '? op = * & x = number & y = number return {"op":"*","x":number,"y":number,"value": x * y}'

pour une division: '? op = / & x = number & y = number return {"op":"/","x":number,"y":number,"value": x / y}'

pour un modulo: '? op = % & x = number & y = number return {"op":"%","x":number,"y":number,"value": x % y}'

pour un factoriel: '? op = ! & n = integer return {"op":"!","n":integer,"value": n!}'

pour savoir si un nombre est premier : '? op = p & n = integer return {"op":"p","n":integer,"value": true if n is a prime number}'

pour savoir la position d'un nombre premier : '? op = np & n = integer return {"op":"np","n":integer,"value": nth prime number}' Merci et bonne journée!
